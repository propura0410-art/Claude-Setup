# SuperApp Repository

This repo is a toolkit for Claude AI to build secure, robust, beautifully designed web applications.

## Structure

```
knowledge/      → Project context, preferences, and guidelines Claude references
templates/      → Starter code and component templates
prompts/        → Workflow definitions and task playbooks
scripts/        → Utility scripts for common tasks
fixtures/       → Sample data and test files
config/         → Security, linting, and project configuration
.github/        → CI/CD automation
```

## Usage

Share this repo URL with Claude and say:
"Clone my repo and use it to build [describe your app]"
