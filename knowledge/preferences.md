# My Preferences

## Writing
- Tone: Clear, friendly, professional
- Format: Clean and scannable
- Audience: General users (non-technical)

## Code
- Languages: TypeScript, Python
- Frontend: React + Next.js + Tailwind CSS
- Backend: Next.js API routes or FastAPI
- Database: PostgreSQL or SQLite
- Auth: NextAuth.js or Supabase Auth
- Testing: Jest + React Testing Library
- Style: Prettier + ESLint (strict)

## Design
- Style: Modern, clean, distinctive — never generic
- Accessibility: WCAG 2.1 AA minimum
- Mobile-first: Always responsive
- Dark mode: Always support both light and dark
- Animations: Subtle, purposeful, performant

## Deployment
- Platform: Vercel (preferred) or Docker
- CI/CD: GitHub Actions
- Environment: .env files, never hardcoded secrets
